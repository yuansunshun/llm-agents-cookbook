## 工具对比

| 工具 | 类别 | 特点 | 适用场景 |
|------|------|------|---------|
| LangChain | Agent 框架 | 组件丰富，生态成熟 | 快速原型，RAG pipeline |
| LlamaIndex | 数据框架 | 专注文档接入和检索 | 多文档问答 |
| AutoGen | 多 Agent | 对话式 Agent 协作 | 复杂任务分工 |
| ChromaDB | 向量数据库 | 本地，零配置 | 开发测试 |
| FAISS | 向量检索 | 高性能，Facebook 出品 | 大规模生产 |
| Pinecone | 向量数据库 | 云托管，托管运维 | 生产部署 |
| LangSmith | 可观测性 | LangChain 链路追踪 | 调试和监控 |
| Weights & Biases | 实验追踪 | 可视化训练曲线 | 模型训练 |
| Ollama | 本地 LLM | 一行命令运行开源模型 | 离线开发 |
| vLLM | 推理引擎 | 高吞吐量 LLM 推理 | 生产部署 |
