# 模块三：实战项目

## 项目列表

### [rag_chatbot/](rag_chatbot/) — RAG 问答机器人

基于 LangChain + ChromaDB 构建的本地知识库问答系统，支持上传 PDF/TXT 文档后进行语义检索问答。

**技术栈**：LangChain · ChromaDB · OpenAI Embeddings · Streamlit（可选）

### [multi_agent_workflow/](multi_agent_workflow/) — 多 Agent 工作流

使用 AutoGen 或 Claude Agent SDK 实现多个专职 Agent 协同完成复杂任务（如：规划 Agent + 执行 Agent + 审核 Agent）。

**技术栈**：AutoGen / Claude Agent SDK · Python

## 运行方式

每个项目目录下有独立的 `README.md`，包含安装步骤和运行命令。
